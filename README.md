# 献立アプリ

## 概要

レシピを登録して一週間分の献立を自動で組んでくれるアプリです。
おすすめのレシピをランダムで追加できる機能もあります。

## 機能

- レシピ作成
- おすすめレシピ追加
- 自動で献立作成
- 食材在庫管理
- 買い物リスト作成
